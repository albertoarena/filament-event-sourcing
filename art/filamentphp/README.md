# filamentphp.com submission assets

Images prepared specifically for the [filamentphp.com plugin listing](https://filamentphp.com/plugins).
They highlight the plugin's functionality (the event-history timeline) with the plugin name and a
short tagline as text, and crop out the panel sidebar/top bar — per the reviewer's feedback.
